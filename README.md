# Sistema de Leilão Online

## Descrição do Projeto

O projeto "Sistema de Leilão Online" é uma aplicação desenvolvida para automatizar e gerenciar as operações de uma casa de leilões. Este sistema permite que usuários registrem itens para leilão, façam lances e acompanhem o status dos leilões em tempo real. O objetivo principal do sistema é facilitar o processo de leilão, proporcionando uma plataforma eficiente, segura e acessível tanto para os organizadores de leilão quanto para os participantes.

## Principais Funcionalidades

1. **Registro de Itens:** Os usuários podem registrar novos itens para leilão, especificando detalhes como nome, descrição, preço inicial, e fotos.
   
2. **Gestão de Lances:** O sistema permite que os participantes façam lances nos itens disponíveis, e atualiza automaticamente o lance mais alto em tempo real.

3. **Notificações:** Os usuários recebem notificações sobre lances feitos em itens de interesse e sobre o encerramento de leilões.

4. **Histórico de Leilões:** O sistema mantém um registro detalhado de todos os leilões realizados, incluindo informações sobre os itens leiloados, lances e vencedores.

5. **Relatórios e Análises:** Os administradores podem gerar relatórios de desempenho e estatísticas dos leilões para análise e tomada de decisões.

## Tecnologias Utilizadas

- **Java:** Utilizado para desenvolver a lógica de negócios e a funcionalidade principal do sistema.
- **MySQL:** Banco de dados utilizado para armazenar informações sobre usuários, itens, lances e histórico de leilões.
